# Selenium C#/.NET Automation Project  
### Automating https://testautomationpractice.blogspot.com/

## 📘 Overview
This project is a complete UI Automation Framework built using **C#**, **.NET**, **Selenium WebDriver**, and **NUnit**.  
The purpose of this project is to automate different web elements available on:

👉 https://testautomationpractice.blogspot.com/

This site contains almost all GUI controls required for real-world automation, including:
- Text fields  
- Radio buttons  
- Checkboxes  
- Dropdowns  
- Multi-select lists  
- Date pickers  
- Alerts  
- Frames  
- Tables  
- Drag and drop  
- Windows handling  
- File upload  
- Progress bar  
- Sliders  
- And more  

This automation framework demonstrates:
- Selenium automation skills  
- Page Object Model (POM)  
- Clean C# coding practices  
- Test design ability  
- Framework structure and reusability  
- Git & GitHub best practices

## 🛠️ Tech Stack
- **C# / .NET 6 (or 8)**
- **Selenium WebDriver**
- **NUnit Test Framework**
- **ChromeDriver**
- **Page Object Model (POM)**
- **Visual Studio 2022**
